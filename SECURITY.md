---

## Reporting a Vulnerability

We take the security of Helix Quantum seriously. If you believe you have discovered a security vulnerability in this project, please follow these steps:

1. **Do NOT open a public GitHub issue.**
2. Send a private report detailing the vulnerability to `security@helix-quantum.dev`.
3. Include the following details in your report:
   - Type of issue (e.g. PIN Auth Bypass, SSE Injection, XSS, Rate Limit Bypass)
   - Steps to reproduce the issue
   - Proof of concept code or payload
   - Impact assessment

---

## Response Timeline

- **Initial Acknowledgment:** Within 24 hours.
- **Triage & Assessment:** Within 48 hours.
- **Fix & Disclosure:** Patch released within 7 days of confirmation.
